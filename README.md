# Гиппократ — Telegram-бот для напоминаний о лекарствах

Гиппократ — это Telegram-бот на Python, который помогает пользователю создавать напоминания о приёме лекарств, просматривать активные напоминания и удалять их.

## Возможности

- Добавление нового напоминания через кнопку или команду `/remind`
- Просмотр списка активных напоминаний
- Удаление напоминаний через inline-кнопку
- Хранение напоминаний в базе данных SQLite
- Автоматическая отправка уведомлений в нужное время

## Используемые технологии

- Python 3
- pyTelegramBotAPI
- SQLite3
- threading
- datetime

## Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/ВАШ_ЛОГИН/ВАШ_РЕПОЗИТОРИЙ.git
```

2. Перейдите в папку проекта:
```bash
cd ВАШ_РЕПОЗИТОРИЙ
```

3. Создайте виртуальное окружение:
```bash
python -m venv venv
```

4. Активируйте виртуальное окружение:

### Windows
```bash
venv\Scripts\activate
```

### Linux / macOS
```bash
source venv/bin/activate
```

5. Установите зависимости:
```bash
pip install -r requirements.txt
```

## requirements.txt

Пример содержимого файла:

```txt
pyTelegramBotAPI
```

Если файла ещё нет, его можно создать вручную или сгенерировать командой:

```bash
pip freeze > requirements.txt
```

## Настройка токена

Перед запуском вставьте токен вашего бота вместо `BOT_TOKEN`:

```python
bot = telebot.TeleBot("BOT_TOKEN")
```

Например:

```python
bot = telebot.TeleBot("ваш_токен_от_BotFather")
```

## Запуск

Запустите бота командой:

```bash
python bot.py
```

## Команды бота

- `/start` — открыть главное меню
- `/hello` — приветствие
- `/restart` — повторно открыть меню
- `/remind` — добавить напоминание
- `/help` — показать список команд

## Как пользоваться

1. Нажмите кнопку **💊 Добавить напоминание**
2. Введите название лекарства
3. Введите дозировку
4. Введите дату и время в формате:
```text
2026-05-26 15:22
```
5. Бот сохранит напоминание и отправит сообщение в нужное время

## Пример

Пример напоминания:

- Лекарство: `Парацетамол`
- Дозировка: `1 таблетка`
- Время: `2026-05-26 15:22`

Сообщение от бота в нужный момент:

```text
💊 ВРЕМЯ ПРИНЯТЬ!
Парацетамол - 1 таблетка
```

## База данных

Бот использует файл `reminders.db`, в котором хранит:

- ID пользователя
- время напоминания
- название лекарства
- дозировку
- статус напоминания (`active` или `sent`)
