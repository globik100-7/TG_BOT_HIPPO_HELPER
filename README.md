# Гиппократ — Telegram-бот для напоминаний о лекарствах

Гиппократ — это Telegram-бот на Python, который помогает создавать напоминания о приёме лекарств.

## Возможности

- Добавление напоминания
- Просмотр активных напоминаний
- Удаление напоминаний
- Хранение данных в SQLite
- Автоматическая отправка уведомлений по времени

## Команды

- `/start` — открыть главное меню
- `/hello` — приветствие
- `/restart` — повторно открыть меню
- `/remind` — добавить напоминание
- `/help` — показать список команд

## Как пользоваться

1. Нажать кнопку **💊 Добавить напоминание**
2. Ввести название лекарства
3. Ввести дозировку
4. Ввести дату и время в формате `YYYY-MM-DD HH:MM`
5. Получить уведомление в указанное время

## Пример

```text
💊 ВРЕМЯ ПРИНЯТЬ!
Парацетамол - 1 таблетка
```

## Описание

Бот сохраняет напоминания пользователя в базу данных и проверяет время каждую минуту. Когда наступает нужный момент, бот отправляет сообщение с названием лекарства и дозировкой.
